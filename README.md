## Setup

Le serveur est basé sur [NodeJS](http://nodejs.org/), et le client est build avec [Grunt](http://gruntjs.com/)

```
  $ git clone https://github.com/KiffanceDeProf/backend.git KiffanceDeProf
  $ cd KiffanceDeProf
  $ npm install
  $ grunt build
  $ node app
```
